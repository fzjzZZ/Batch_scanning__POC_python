# FOFA

## FOFA pro API

`FOFA Pro API`是资产搜索引擎FOFA Pro为开发者提供的`RESTful API`接口，允许开发者在自己的项目中集成`FOFA Pro`的功能。

API手册链接 :https://fofa.so/api

## 环境

python 3

## 依赖库安装

` pip install pyfofa`

`pip install argparse`

## 使用说明

1. 安装依赖库

2. 将fofa账号的登陆邮箱以及api写入到脚本中
